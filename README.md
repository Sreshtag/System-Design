# 🚀 System Design Simulator

An interactive **System Design Learning Tool** built with **React + React Flow** that helps users practice designing scalable systems visually.

Users can drag & drop architecture components, connect them on a canvas, and get instant feedback based on their design.

---

## ✨ Features

* 🎯 Prebuilt design problems:

  * URL Shortener
  * WhatsApp
  * Instagram
  * Uber

* 🧩 Drag & Drop Components:

  * Client
  * Load Balancer
  * DNS
  * CDN
  * API Server
  * Database
  * Cache
  * Queue

* 🔗 Connect components visually using React Flow

* 📊 Scoring System

  * Checks required components
  * Validates important architecture connections

* 💡 Smart Feedback

  * Explains why missing components matter
  * Problem-specific suggestions

* 🌙 Modern Dark UI

---

## 🛠️ Tech Stack

* React (Vite)
* React Flow
* Tailwind CSS

---

## ▶️ Run Locally

```bash
npm install
npm run dev
```

---

## 📸 How It Works

1. Select a system design problem
2. Drag components to the canvas
3. Connect them logically
4. Click **Submit**
5. View your score and feedback
